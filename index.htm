AWC File Browser

ファイルを整理する時に、中身を確認してから(After watching the contents)捨てるか残すかを判断できるファイルブラウザ―です。
  • 左半分はファイルブラウザ
    • ドラッグ＆ドロップ操作のコピー、カット、ペーストにも対応したTree構造
    • 右クリックのメニューで不要なファイルはゴミ箱に移動
    • その他、新規ホルダの作成や名称変更、フォルダ丸ごとのコピー
    • このアプリで表示できなければ割り付けられたアプリを起動…
    といったファイルブラウザとしては当たり前の機能を装備
  • 右半分はWebベースのメディアプレイヤー
    • Windows Media Player12相当の組み込みプレイヤーで.m2ts(ビデオカメラの出力ファイル)なども表示できます。
    • WebベースなのでWindows Media Playerで再生できないファイルはプラグインを組み込んだページを生成して対応します。
    • webはEdge相当なのでHTML5のvideo/audioタグも使って再生します。
最新版は2017年9月20日（Ver,1.2.0)汎用プレイリスト(.m3u)対応
  • 自動送りは出来ませんのでボタンクリック操作して下さい。

tex2calc (文字cal win)

コピーしてきたテキストから数字と演算子、関数を抜き出して計算します。
  • 左半分は入力TextBoxと計算経過表示
    • 計算間違いを見つけたらそこだけ再入力して再計算
    • 計算経過表示でお子さんに正しい計算順を教える時にも使えます。
  • 右半分は計算機としてのソフトキーボード
    • タッチパネルで当たり前の電卓操作
    • デザインの切り替えも用意します。

減算時計 (仮名)

スケジュールやアラームに設定した時刻までの残り時間を表示します。
  • 目覚ましアラームと連携して夜更かし防止に
    • 実質的な目覚まし時計に使われているスマホのアラームと連携する方法を模索しています。
  • スケジュールと連携してアポ忘れ防止
  • シンプルにタイマーとしてもご利用頂けます。

当たり前に使える音楽プレイヤー
MaraSongs

「延々続くこと」を示す言い回しの一つにmarathonという単語が出てきた事があり、延々と再生を続けるミュージックプレイヤーの名前としてこの造語を思いつきました。
たとえマラソンランナーがBGMに使っても、立ち止ってアルバムの選択をし直さなくても良いプレイヤーを作成しました。

機能的にはi-Podクラシックの代用になる様、開発に着手
  • AAC/MP3の歌詞表示
  • 全曲連続再生
にも対応。また
  • イコライザー
  • リバーブ
  • ノティフィケーションプレイヤー
などAndroidが用意しているミュージックプレイヤー機能も一通り網羅しました。
最新版は2016年5月8日（Ver,1.2.47)

スマホの大画面を活かす電卓
文字cal

文字で数式を書き込んで計算させる電卓です。数字も演算子もそのまま残るから
  1. 変更したい部分だけを修正して再計算が可能。
  2. 誤入力もすぐ判明
出来る計算は今の処、
  • 小学生の算数
  • 「一般向け」の計算
使い方は(1+2)/+3などの数式を入力して”＝”を入力した時点で答えにたどり着くまでを表示します。
最新版は2016年5月8日(1.4.1)Android 5.1対応

カンタン・速攻で電話できる
Phone2nd

電話帳、発信・着信履歴と電話をかけるためのユーティリティをまとめました。
機能的には
  • 通話を始めるまでのステップ削減。
    例えば起動時はかけ直しの頻度の高い最新履歴で起動
  • ガラケーのスピードダイヤル代わりの"定番リスト"
  • 履歴はお名前でまとめて表示。
    発信/着信などは履歴として表示。
  • 電話帳はAndroidで用意している全フィールドが利用可能
  • 実際に回線を接続する画面はスマホに搭載されているものをそのまま使う事で通信キャリアを問わず、ご利用いただけます。
最新版は1.3.2; 2016年5月22日；Android2.3互換パートが残っていたために発生していたバグの修正

シンプルな乱数発生器
アタレくん

連続して同じ数字が出ないアルゴリズム”を求めて開発を続けてる乱数発生器です。

ただ乱数を発生させるのも退屈なので世界のロトくじの法則で乱数発生させる事もできます。
ご注意
  1. 予想の様な複雑な事は出来ませんが、数字が思いつかない時にお役にたてれば幸いです。
  2. 宝くじは販売地域の地方税財源です。他国の宝くじを販売地区以外で売買すると法律違反とされる事が殆どですのでご注意ください。
最新版は2016/8/26（Ver1.4.1)
