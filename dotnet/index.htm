AWC File Browser

概要
ファイルを整理する時に、中身を確認してから(After watching the contents)捨てるか残すかを判断できるファイルブラウザ―です。


最新版は 2017年10月14日(Ver,1.3.0)ファイルブラウザの２ペイン化
ダウンロード
awsfilebrowser.zip
の中の「setup.exe」をダブルクリックするとインストールします
警告が発生するかもしれません
  • ブラウザのダウンロードでは破棄/継続の「継続」を選択して下さい。 (ご参照)
  • Windows Defender SmartScreenのインストールブロックに引っ掛かりますが、「詳細情報」をクリックするとインストールを継続できます。 (ご参照)
正しい配布方法を確認中です。

tex2calc (文字cal win)

只今考案中
概要
コピーしてきたテキストから数字と演算子、関数を抜き出して計算します。
  • 左半分は入力TextBoxと計算経過表示
    • 計算間違いを見つけたらそこだけ再入力して再計算
    • 計算経過表示でお子さんに正しい計算順を教える時にも使えます。
  • 右半分は計算機としてのソフトキーボード
    • タッチパネルで当たり前の電卓操作
    • デザインの切り替えも用意します。
